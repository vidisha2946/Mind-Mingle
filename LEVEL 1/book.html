1. Basic Data Types:

INT: Used to store integers (whole numbers), both positive and negative.
Size: Typically 4 bytes (depends on the system).
Format Specifier: %d

FLOAT: Used to store single-precision floating-point numbers (decimal numbers).
Size: Typically 4 bytes (32 bits).
Format Specifier: %f

DOUBLE: Used to store double-precision floating-point numbers.
Size: 4 bytes.
Format Specifier: %lf;

CHAR: Used to store a single character.
Size: 8 bytes.
Format Specifier: %c

LONG:It is typically used when the range of values required exceeds what can be stored in a regular integer
Size: 32 bytes
Format Specifier: %ld
