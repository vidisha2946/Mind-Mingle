3. Overview of Conditional Statements:

General Overview of Conditional Statements: Conditional statements control the flow of execution in a program by allowing the program to make decisions based on conditions.

If-Else Statement: An `if-else` statement provides an alternative block of code to execute if the condition is false.

Syntax Of If-Else Statement in C language:
if (condition) {
// code block if true
} else {
// code block if false
}
Switch Statement:
A `switch` statement selects one of many blocks of code to execute based on the value of an expression.

Syntax Of Switch Statement in C language:
switch (expression) {
case value1:
// code block for value1
break;
case value2:
// code block for value2
break;
default:
// code block if no case matches
}
