2. Overview of Loops:

General Overview of Loops:
Loops are fundamental programming constructs that allow you to execute a block of code repeatedly based on certain conditions. They help reduce redundancy and make code more efficient and manageable.

While Loop:
A while loop repeatedly executes a block of code as long as a specified condition evaluates to true.

Syntax Of While Loop in C language:
while (condition) {
// code block to be executed
}

for loop:
for Loop A for loop iterates over a sequence (such as a list, tuple, or range) or other iterable objects.

Syntax Of For Loop in C language:
for (initialization; condition; update) {
// code block to be executed
}
